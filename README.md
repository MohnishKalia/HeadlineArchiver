# Client Quickstart

```console
cd client/
```

## Run locally

```console
npm run start
```

# Scraper Quickstart

```console
cd scraper/
```

Add [firebase admin private key](https://console.firebase.google.com/project/headline-archiver/settings/serviceaccounts/adminsdk) into directory

## Run locally

Open `index.ts` and edit puppeteer's chromium executable path

```console
npm run start
```

## Package for export

```console
npm run pkg
```
